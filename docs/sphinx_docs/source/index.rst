ECOGEN's documentation
======================

Here is described the documentation of the ECOGEN_ open source plateform dedicated to numerical simulation of compressible multiphase flows.

ECOGEN is an open source project distributed under under `GNU GPLv3 License`_ and available on `Git-Hub`_.

It is developped by code-mphi members in partenrship with AMU_, Caltech_ and CNES_.

In this document, ECOGEN's user will find all necessary information to use the software.

Interested developpers are encouraged to contribute in the project and can use the API documentation generated by Doxygen_.
